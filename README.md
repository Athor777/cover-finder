## Come metterla sulla schermata Home (iPhone)

1. Apri il link qui sopra in **Safari**
2. Tocca il pulsante **Condividi** (quadrato con freccia)
3. Scorri e tocca **"Aggiungi alla schermata Home"**
4. Conferma → diventa un'icona come un'app

## Cosa contiene

- ~165 brani iconici di videogiochi
- 6 decenni: anni 70, 80, 90, 2000, 2010, 2020
- Filtri per decennio e per stato (Da fare / In corso / Pubblicate)
- Ricerca testo libero
- Pulsante YouTube per ascoltare ogni brano
- Stato salvato sul dispositivo
